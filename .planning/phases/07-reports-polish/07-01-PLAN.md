<objective>
Build the Reports API: velocity chart data, burndown chart data, project KPIs, and CSV export.

Purpose: Reports provide visibility into team performance and project health.
Output: API returns chart-ready data and exportable backlog.
</objective>

<tasks>

<task type="auto">
  <name>Task 1: Velocity and burndown endpoints</name>
  <action>
1. `VelocityView` (APIView):
   - GET: returns last 5 completed sprints with:
     - sprint_name, planned_points, completed_points
   - Includes average_velocity calculation

2. `BurndownView` (APIView):
   - GET with sprint_id: returns daily data for sprint:
     - For each day from start_date to end_date (or today):
       - date, ideal_remaining, actual_remaining
     - ideal_remaining: linear decrease from total to 0
     - actual_remaining: total - sum(completed points up to that day)
   - Based on TaskHistory (when tasks moved to done column)

3. `DashboardKPIView` (APIView):
   - GET: returns project-level KPIs:
     - total_tasks, completed_tasks, in_progress_tasks, blocked_tasks
     - average_velocity (from completed sprints)
     - current_sprint_progress_percentage
     - team_size
  </action>
</task>

<task type="auto">
  <name>Task 2: CSV export endpoint</name>
  <action>
`ExportCSVView` (APIView):
- GET: generates CSV with all backlog items
- Columns: Clave, Título, Tipo, Prioridad, Estado, Sprint, Asignado, Story Points, Etiquetas, Creado
- Returns as file download: `{project_key}_backlog_{date}.csv`
- Content-Type: text/csv
- Content-Disposition: attachment
  </action>
</task>

<task type="auto">
  <name>Task 3: Frontend API</name>
  <action>
```typescript
export const reportsApi = {
  getVelocity: (projectId: string) =>
    client.get<VelocityData[]>(`/reports/${projectId}/velocity/`),
  getBurndown: (projectId: string, sprintId: string) =>
    client.get<BurndownData[]>(`/reports/${projectId}/burndown/?sprint_id=${sprintId}`),
  getDashboardKPIs: (projectId: string) =>
    client.get<DashboardKPIs>(`/reports/${projectId}/dashboard/`),
  exportCSV: (projectId: string) =>
    client.get(`/projects/${projectId}/backlog/export/csv/`, { responseType: 'blob' }),
};
```
  </action>
</task>
