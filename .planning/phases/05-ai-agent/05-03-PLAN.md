3. Context builder function:
   ```python
   def build_project_context(project_id):
       """Build context string for AI with real project data."""
       project = Project.objects.get(id=project_id)
       active_sprint = project.sprints.filter(status='active').first()
       tasks = project.tasks.all()
       return f"""
       Proyecto: {project.name} ({project.key})
       Sprint activo: {active_sprint.name if active_sprint else 'Ninguno'}
       Tareas totales: {tasks.count()}
       Por hacer: {tasks.filter(column__name='Por Hacer').count()}
       En progreso: {tasks.filter(column__name='En Progreso').count()}
       Completadas: {tasks.filter(column__is_done_column=True).count()}
       Story points sprint: {sprint_points_info}
       Miembros: {members_list}
       Última actividad: {recent_changes}
       """
   ```
