Custom hooks:
- `useGenerateStories(projectId)`: submit → poll every 2s → return results
- `useGenerateBacklog(projectId)`: same pattern for full backlog
- `useSuggestPriorities(projectId)`: same pattern
- `useChat(projectId)`: send message, get response, manage conversation history
- `useAIRecommendations(projectId)`: fetch contextual recommendations

Polling pattern:
```typescript
const pollForResult = async (taskId: string) => {
  while (true) {
    const { data } = await aiApi.getTaskStatus(projectId, taskId);
    if (data.status === 'completed') return data.result;
    if (data.status === 'failed') throw new Error('AI task failed');
    await new Promise(r => setTimeout(r, 2000));
  }
};
```
