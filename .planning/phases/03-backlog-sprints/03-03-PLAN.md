<task type="auto">
  <name>Task 2: Sprint views</name>
  <files>backend/boards/views.py (append), backend/boards/urls.py (append)</files>
  <action>
`SprintViewSet`:
- `list`: sprints for project, ordered by created_at desc
- `create`: auto-name "Sprint N" (N = count + 1), set status='planning'
- `start` (custom action): validates no other active sprint, sets status='active' + start_date
- `complete` (custom action):
  - Sets status='completed', completed_at=now
  - Based on incomplete_action:
    - 'backlog': unassign incomplete tasks (sprint=null, column=null)
    - 'next_sprint': create new sprint and assign incomplete tasks to it
  - Tasks in is_done_column remain with completed sprint
- `assign` (custom action): assign list of task IDs to this sprint + set column to first column
- Permissions: only Owner/Admin can start/complete sprints

URL: `projects/{pid}/sprints/` → SprintViewSet
  </action>
</task>
