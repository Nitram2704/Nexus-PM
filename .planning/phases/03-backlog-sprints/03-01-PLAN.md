<task type="auto">
  <name>Task 2: Task views with filtering</name>
  <files>backend/boards/views.py, backend/boards/urls.py, backend/boards/filters.py</files>
  <action>
1. `TaskViewSet` (ModelViewSet):
   - Scoped to project: `GET /api/v1/projects/{pid}/backlog/`
   - List: filtered by type, priority, assignee, sprint, label, search text
   - Create: auto-sets project, auto-generates number
   - Update: tracks changes in TaskHistory
   - Custom action `reorder`: POST with array of {id, order} to batch reorder
   - Custom action `export_csv`: GET to export filtered tasks as CSV

2. `TaskFilter` (django-filter):
   - type, priority, assignee, sprint (include unassigned), labels, search (title + description)

3. `CommentViewSet`: nested under task
   - `GET/POST /api/v1/projects/{pid}/backlog/{tid}/comments/`

4. URL patterns:
   ```python
   projects/{pid}/backlog/ → TaskViewSet
   projects/{pid}/backlog/{tid}/comments/ → CommentViewSet
   projects/{pid}/labels/ → LabelViewSet
   ```
  </action>
</task>
