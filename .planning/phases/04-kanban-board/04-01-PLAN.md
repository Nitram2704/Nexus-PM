<objective>
Build the Board API: get board state (columns + cards), move card between columns, column CRUD with reorder and WIP limits.

Purpose: The API that powers the Kanban board view. Must be fast and support optimistic updates.
Output: Board endpoint returns complete state, move endpoint is atomic and fast.
</objective>

<tasks>

<task type="auto">
  <name>Task 1: Board serializers</name>
  <action>
- `BoardSerializer`: list of columns, each with nested tasks (cards). Includes column name, order, wip_limit, task count.
- `ColumnSerializer`: id, name, order, wip_limit, is_done_column
- `ColumnCreateSerializer`: name, order, wip_limit
- `CardMoveSerializer`: task_id, target_column_id, order (position within column)
- `ColumnReorderSerializer`: list of { id, order } pairs
  </action>
</task>

<task type="auto">
  <name>Task 2: Board views</name>
  <action>
1. `BoardView` (APIView):
   - `GET /projects/{pid}/board/`: Returns all columns with their tasks for the active sprint. Tasks filtered by sprint=active. Ordered by column.order, then task.order.

2. `ColumnViewSet`:
   - CRUD for columns
   - `reorder` action: batch update column orders
   - Permission: Owner/Admin for create/delete, member for read

3. `CardMoveView` (APIView):
   - `POST /projects/{pid}/board/cards/move/`: Moves a task to target column + order
   - Updates task.column and task.order
   - Returns updated task
   - Logs move in TaskHistory
  </action>
</task>

<task type="auto">
  <name>Task 3: Frontend API client</name>
  <action>
```typescript
export const boardApi = {
  getBoard: (projectId: string) => client.get<BoardState>(`/projects/${projectId}/board/`),
  moveCard: (projectId: string, data: { task_id: string; target_column_id: string; order: number }) =>
    client.post(`/projects/${projectId}/board/cards/move/`, data),
  createColumn: (projectId: string, data: { name: string; wip_limit?: number }) =>
    client.post(`/projects/${projectId}/board/columns/`, data),
  updateColumn: (projectId: string, columnId: string, data: Partial<Column>) =>
    client.put(`/projects/${projectId}/board/columns/${columnId}/`, data),
  deleteColumn: (projectId: string, columnId: string) =>
    client.delete(`/projects/${projectId}/board/columns/${columnId}/`),
  reorderColumns: (projectId: string, items: { id: string; order: number }[]) =>
    client.post(`/projects/${projectId}/board/columns/reorder/`, { items }),
};
```
  </action>
</task>
