<task type="auto">
  <name>Task 1: Board state management</name>
  <files>frontend/src/hooks/useBoard.ts, frontend/src/stores/boardStore.ts</files>
  <action>
1. `boardStore.ts`: Zustand store for board state
   - Holds current board data (columns + cards)
   - `moveCard(taskId, sourceCol, destCol, destIndex)`: optimistic reorder
   - `revertMove()`: undo last move on API failure

2. `useBoard.ts`: TanStack Query hook
   - `useBoard(projectId)`: fetches board state, syncs to boardStore
   - `useMoveCard()`: mutation with optimistic update pattern:
     1. Instantly update boardStore (UI moves card)
     2. Call API in background
     3. On error: revert boardStore + show error toast
  </action>
</task>
