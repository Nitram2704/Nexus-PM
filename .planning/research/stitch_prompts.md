# Prompts for Google Stitch (Nexus-PM)

You can copy and paste the following prompts directly into **Google Stitch** to generate the UI mockups for this project.

## 1. Project Dashboard (Dark Mode con Personalidad)

```text
Project management dashboard "Command Center" for software teams.
STRICT DARK MODE ONLY. Evita diseños genéricos de plantillas SaaS corporativas.
Queremos una personalidad única, técnica y de alto impacto (Tech-forward / Cyber-Agile). 
Usa fondos charcoal casi negros (#0D1117) y haz que las tarjetas se sientan exclusivas usando bordes finos luminosos, asimetría sutil y diseño vanguardista.
Acentos principales en Electric Blue vibrante y destellos de Warm Amber. Muestra la tipografía JetBrains Mono de forma agresiva para los datos.

Key Features:
- Top header with search bar, notification bell (with unread badge), and user avatar
- Welcome section with "Proyectos Recientes" and "Nuevo Proyecto" action
- Grid of Project Cards. Each card shows: Project Name, Key (e.g., NEX), description snippet, team avatars, and a progress bar for the active sprint
- Left collapsible sidebar with navigation icons
```

## 2. Interactive Kanban Board (Dark Mode con Personalidad)

```text
Interactive Kanban Board screen for an agile project management app.
STRICT DARK MODE ONLY. Aléjate del diseño SaaS tradicional. Necesita personalidad de "Centro de Control Misión Crítica".
Tarjetas Kanban con estética glassmorphic oscura, fondos casi negros y sombras tipo neón muy sutiles para indicar elevación. Usa tonos Electric Blue para las barras de progreso.

Key Features:
- Top sprint header showing: Sprint Name, days remaining countdown, and "Completar Sprint" button
- 4 vertical columns: "To Do", "In Progress", "In Review", "Done"
- Column headers include a task count and a WIP (Work In Progress) progress bar
- Kanban cards showing: Task Key (e.g., NEX-015), Task Title, Priority color indicator on left border, User Avatar, and Story points badge
- A contextual "AI Scrum Master" floating bubble action button in the bottom right corner con un brillo intenso Warm Amber (#F59E0B) para diferenciarlo de la UI estandar.
```
